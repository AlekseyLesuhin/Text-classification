# Text-classification
Этот обучающий проект наглядно демонстрирует все этапы решения классической NLP задачи классификации текста. В качестве примера взята задача классификации отзывов гостей ресторанов на позитивные и негативные. Набор данных предоставляет собой датасет из отзывов пользователей о ресторане и меткой: 1 - позитивный отзыв, 2 - негативный отзыв.

# Данные
Размер: 22163 записи
Признаки:

    text - отзыв гостя

Целевая переменная: 
    
    label (0/1)

Особенности данных:
    
    Сбалансированный датасет

# Решение
1. EDA (Исследовательский анализ данных)

    Анализ структуры и качества данных
    Исследование распределения длины текстов
    Анализ важности токенов для классификации
    Выявление ключевых слов и фраз

2. Baseline модели

    CountVectorizer + классические ML алгоритмы:
        LogisticRegression
        DecisionTreeClassifier
        RandomForestClassifier
        LGBMClassifier
    TF-IDF векторизация
    Очистка данных: приведение к нижнему регистру, удаление лишних пробелов

3. Нейронные сети

    RNN - рекуррентная сеть для анализа последовательностей

    CNN - сверточная сеть для выявления локальных паттернов

    LSTM - долгосрочная память для сложных зависимостей

    LSTM + GloVe - предобученные эмбеддинги слов
